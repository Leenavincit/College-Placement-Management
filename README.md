# 🎓 College Placement Management System

## 📌 Project Overview

The **College Placement Management System** is a web-based application developed to automate and simplify the campus recruitment process. It provides a centralized platform for administrators, students, and recruiting companies to manage placement activities efficiently. The system reduces manual work, improves communication, and streamlines the recruitment process through a secure and user-friendly interface.

---

## ✨ Features

### 👨‍💼 Admin Module
- Admin Login Authentication
- Manage Student Records
- Manage Company Details
- Post Job Opportunities
- View Applications
- Monitor Placement Activities

### 👩‍🎓 Student Module
- Student Registration & Login
- Update Profile
- Upload Resume
- View Available Jobs
- Apply for Jobs
- Track Application Status
- View Interview Call Letters

### 🏢 Company Module
- Company Registration & Login
- Post Job Openings
- View Applicants
- Shortlist or Reject Candidates
- Schedule Interviews
- Send Interview Call Letters

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Leenavincit/College-Placement-Management.git
```

2. Move the project folder to the **htdocs** directory in XAMPP.

3. Start **Apache** and **MySQL**.

4. Import the SQL database into **phpMyAdmin**.

5. Open your browser and visit:

```
http://localhost/College-Placement-Management/
```
